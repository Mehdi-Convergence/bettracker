# BetTracker - Proposition de contenu par plan

## Positionnement

- **Free** : Decouverte complete pendant 7 jours, puis acces minimal
- **Pro** : L'essentiel pour un parieur serieux (scanner + analyse + suivi)
- **Elite** : Automatisation complete + IA + support premium

## Proposition detaillee

### FREE (0 EUR — 7 jours d'essai)

**Objectif** : Faire decouvrir la puissance de l'outil pour convertir en Pro/Elite

**Pendant le trial (7 jours)** :
- Acces complet a TOUS les modules (identique Elite)
- Le but : que l'utilisateur ne puisse plus s'en passer

**Apres le trial** :
- Scanner : consultation du dernier scan cache uniquement (pas de refresh force)
- Dashboard : lecture seule, pas de stats detaillees
- Portfolio : consultation uniquement, pas d'ajout de paris
- Backtest : non accessible
- Campagnes : non accessible
- IA Analyste : 5 msg/jour (teaser)
- Pas d'export CSV

### PRO (29 EUR/mois — 23 EUR/mois en annuel)

**Objectif** : L'outil quotidien du parieur value bet

**Inclus** :
- Scanner IA illimite (tous sports, tous timeframes)
- Dashboard complet avec stats detaillees
- Portfolio : ajout/suivi/settlement de paris, CLV auto
- Backtest : tous sports, toutes strategies, sauvegardes illimitees
- Partage de tickets (lien public)
- Notifications (in-app + email pour les paris settles)
- IA Analyste : 50 msg/jour
- Export CSV du portfolio

**Non inclus** :
- Campagnes (automatisation)
- Combos (generateur)
- Support prioritaire

### ELITE (69 EUR/mois — 55 EUR/mois en annuel)

**Objectif** : L'automatisation totale — l'algo travaille pour vous

**Tout le Pro, plus** :
- Campagnes illimitees (filtres custom, recommandations auto, version control)
- Generateur de combos (optimisation multi-match)
- IA Analyste : 200 msg/jour
- Support prioritaire (reponse < 24h)
- Acces prioritaire aux nouvelles features (beta)
- Export CSV avance (historique complet)

## Metriques a suivre pour ajuster les plans

1. **Taux de conversion trial -> Pro** : cible > 15%
2. **Taux de conversion Pro -> Elite** : cible > 10%
3. **Churn mensuel** : cible < 8%
4. **Usage IA par tier** : si les free epuisent systematiquement le quota, c'est bon signe
5. **Modules les plus utilises** : pour savoir quoi mettre en avant
6. **Revenue par user (ARPU)** : cible > 35 EUR/mois (mix Pro + Elite)

## Evolution possible des plans

### Court terme
- Ajouter un plan "Starter" a 9 EUR/mois (scanner football only, 1 backtest/jour)
- Offre annuelle avec 2 mois gratuits

### Moyen terme
- Plan "Team" pour les groupes (partage de campagnes, leaderboard interne)
- API access pour les devs (webhooks, custom integrations)
- Plan "Agency" pour les tipsters pro (white-label, multi-client)

### Long terme
- Marketplace de strategies (les Elite partagent leurs campagnes gagnantes)
- Copy-trading (suivre les paris d'un tipster verifie)
- Odds monitoring en temps reel (alertes push quand un edge apparait)
